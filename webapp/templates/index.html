// Model Architecture

How It Works

🧠
BACKBONE
EfficientNet-B3
Pretrained on ImageNet. Compound scaling optimizes depth, width, and resolution simultaneously for the best accuracy/compute tradeoff.
12M Parameters
224px Input size
🎯
CLASSIFIER HEAD
Custom MLP + BN
Two-layer MLP with BatchNorm and SiLU activation. Dropout(0.4) prevents overfitting. Kaiming initialization for optimal gradient flow.
10 Classes
0.4 Dropout
🔍
EXPLAINABILITY
Grad-CAM
Gradient-weighted Class Activation Mapping highlights which image regions drove the prediction — verifying the model attends to hands, phone, and face.
Last Conv Layer
⚖️
TRAINING
Two-Phase Fine-Tuning
Phase 1: train head only. Phase 2: unfreeze backbone with 10× lower LR. Label smoothing + WeightedRandomSampler for class imbalance.
